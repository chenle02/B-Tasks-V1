# B-Tasks V1

Task app for homework, chores, and ideas.

Features:
- Add/edit/delete tasks
- Checkboxes
- Priority: Low / Normal / High
- Due date
- Search
- Filters: All / Today / Open / Done / High
- Clear completed tasks
- Local storage
- Shortcuts to B-Launcher, B-Notes, B-Map, and B-Translate
- GitHub Actions APK build
